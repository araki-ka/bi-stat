# bi-stat

## 目的
BIクラフト(仮名)の構成管理の演習用リポジトリです。

## テーマ
- Tableauを用いて、政府統計データ(e-Stat)をの可視化を行う。
- ただし、e-statのデータは正規化されていないので、データの正規化を行い、Tableauで可視化しやすい形にする。
